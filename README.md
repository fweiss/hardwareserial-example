# uart
Sample program using 3 UARTs on Sparkfun ESP32 Thing

## Hookup pins
Hookup the board to the oscilloscope as follows:
- GND
- TX0 = TX/GPIO1 = channel 1 yellow
- TX1 = GPIO25 = channel 2 violet
- TX2 = GPIO17 = channel 3 blue
